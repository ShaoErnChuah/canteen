Canteen Ordering System 
An interactive online canteen ordering system designed for students and staff to browse available food items, place orders, and receive confirmations instantly.

Overview

The Canteen Ordering System is a web-based application built to digitalize the food ordering process at INTI College Penang.
It provides:
-Online food ordering
-Real-time product availability
-Admin & user access levels
-Automated email verification and order confirmation
-fully functional MySQL-backed database
-Secure login with hashed passwords
-Student & staff account creation

OTP-based email verification

Developed using PHP, MySQL, ,PHPMailer, HTML/CSS/JS and using XAMPP as its local enviornment 


Student and staff can:
-View available canteen items
-Check real-time quantity & availability
-Place orders quickly
-Receive order details through email

Admin Functions:
Manage product inventory (add, edit, update quantity / price)
Track sales history
View user orders
Full administrative control through an admin dashboard



This system was developed by Chuah Shao Ern as part of the Final Year Project for INTI College Penang in 2023
